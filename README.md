The main objective of this project is to develop a simple Attendance Management System using Python and NumPy that records and analyzes attendance data efficiently.

This project aims to:

✔️ Collect attendance data for multiple persons over a given number of days.

✔️ Store the attendance information in a structured format (matrix form).

✔️ Convert the data into a NumPy array for efficient numerical operations.

✔️ Calculate total present and absent days for each person.

✔️ Compute attendance percentage automatically.

✔️ Analyze attendance trends day-wise and person-wise.

✔️ Build a foundation for developing a full Attendance Management System in the future.

✔️ Reduce manual calculation errors by automating attendance analysis.
